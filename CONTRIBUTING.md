# Contributing to adb-web

Thanks for your interest in contributing! 🎉

## Getting Started

1. Fork the repo
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/adb-web.git`
3. Install dependencies: `npm install`
4. Run locally: `npm start`
5. Open `http://localhost:3737`

## Making Changes

- Keep the stack simple — **no new frameworks** unless really necessary
- Test with a real Android device connected via USB
- Make sure existing features still work after your change

## Submitting a PR

1. Create a branch: `git checkout -b feature/your-feature`
2. Commit with a clear message: `git commit -m "feat: add wireless ADB support"`
3. Push: `git push origin feature/your-feature`
4. Open a Pull Request on GitHub

## Commit Message Style

Use emoji prefixes for clarity:

| Prefix | When to use |
|--------|-------------|
| `🚀 feat:` | New feature |
| `🐛 fix:` | Bug fix |
| `📝 docs:` | Documentation |
| `🎨 style:` | UI/CSS changes |
| `♻️ refactor:` | Code refactor |
| `⚡ perf:` | Performance |
